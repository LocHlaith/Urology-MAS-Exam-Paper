## 重大命题缺陷（major defects）

重大命题缺陷（major defects） 指实质性威胁题目医学正确性、可回答性、评分效度、考试蓝图一致性、公平性、临床安全性或盲法来源辨识效度的题目层面缺陷。重大命题缺陷包括 critical defects（关键/致命缺陷） 和 moderate defects（中度缺陷）以及影响作答的Technical/Formatting Defects（技术/格式缺陷）（如选项编号缺失或重复，选项显示顺序与答案键不一致，题型标签错误等）。Critical defects 代表完全不可接受的缺陷；moderate defects 代表程度不如Critical defects那么严重，经过进一步更改修订后可能仍可以被使用的错误。只有当格式错误会影响考生能否公平、明确、唯一地作答，或影响评分和研究设计完整性时，才应算作 fatal Technical/Formatting Defects。鉴于本研究要验证UroEmas系统的出题效果，因此只要出现任一major defect不得进入最终试卷。

## Critical defects

| 编号 | 建议纳入的 Critical defect | 操作性判定标准 | 主要参考依据 |
| --- | --- | --- | --- |
| C1 | 无正确答案 | 所有选项均不符合指定教材、指南或泌尿外科共识；考生无法通过专业知识选出正确答案。 | Downing 2005；NBME Item-Writing Guide；Haladyna et al. 2002 |
| C2 | 正确答案标注错误 | 题目存在正确答案，但答案键标注为错误选项，导致评分错误。 | Downing 2005；NBME Item-Writing Guide；AERA/APA/NCME Standards |
| C3 | 单选题存在多个同等正确答案 | A1、A2、A3/A4 或 B 型单选题中，两个或以上选项均可被合理认为正确，违反 one-best-answer 原则。 | NBME Item-Writing Guide；Case & Swanson / NBME manual；Haladyna et al. 2002 |
| C4 | 题干与正确答案逻辑矛盾 | 病例信息、检查结果或设问逻辑支持的答案与标注答案不一致，或题干内部前后矛盾。 | Downing 2005；NBME Item-Writing Guide；Tarrant et al. 2006 |
| C5 | 核心医学事实错误 | 涉及诊断、治疗、分期、指南推荐、禁忌证、急症处理、药物使用或并发症处理等核心知识错误。 | Downing 2005；AERA/APA/NCME Standards；UroEMAS protocol |
| C6 | 严重偏离考试蓝图或考查范围 | 题目明显不属于泌尿外科住培考试范围，或与预设题型、考点、认知层级严重不一致。你的方案明确人工题和 UroEMAS 题均应遵循同一考试蓝图。 | AERA/APA/NCME Standards；Haladyna et al. 2002；UroEMAS protocol |
| C7 | 严重公平性、偏倚或歧视问题 | 题目包含基于性别、年龄、地域、职业、经济状况、文化背景等不当假设，或非医学背景因素决定作答结果。 | AERA/APA/NCME Standards；Downing 2005；UroEMAS protocol |
| C8 | 评分或考试系统错误导致结果不可用 | 答案键错位、选项导入错误、题目显示缺失、题号与答案表不匹配，导致无法可靠评分。 | AERA/APA/NCME Standards；NBME Item-Writing Guide；UroEMAS protocol |
| C9 | A3/A4 连续病例题内部断裂 | 同一病例下子题的病情演进、检查结果、诊断前提或治疗阶段相互矛盾，导致后续子题不可解。 | NBME Item-Writing Guide；Case & Swanson / NBME manual；UroEMAS protocol |
| C10 | X 型题正确答案组合不可判定 | 不定项选择题中多个选项“可能正确但非必选”，或正确答案组合无法由题干唯一推出。 | Haladyna et al. 2002；Tarrant et al. 2006；UroEMAS protocol |

## Moderate defects

| 编号 | 建议纳入的 Moderate defect | 操作性判定标准 | 主要参考依据 |
| --- | --- | --- | --- |
| M1 | 题干信息不足 | 病例型题缺少必要临床信息，例如年龄、性别、主诉、阳性症状、体征、关键检查或病程，导致考生需猜测关键前提。 | NBME Item-Writing Guide；Case & Swanson / NBME manual；UroEMAS protocol |
| M2 | 题干信息冗余或无关信息过多 | 题干堆砌大量与作答无关的信息，增加阅读负担并引入构念无关变异。 | Downing 2005；Haladyna et al. 2002；Tarrant et al. 2006 |
| M3 | 题干线索泄露答案 | 题干中出现与正确选项高度同义或直接提示答案的关键词，使题目主要考查文字匹配而非医学判断。 | Haladyna et al. 2002；Tarrant et al. 2006；Downing 2005 |
| M4 | 认知层级偏离预设目标 | 预设为临床推理题但实际只考记忆；或预设为基础知识题却需要复杂病例综合判断。你的模型计划纳入 cognitive_level，因此应记录该类缺陷。 | AERA/APA/NCME Standards；Haladyna et al. 2002；UroEMAS protocol |
| M5 | 答案解析不能充分支持答案 | 解析与正确答案不一致、理由不足，或不能解释关键干扰项为什么错误。 | NBME Item-Writing Guide；AERA/APA/NCME Standards；UroEMAS protocol |
| M6 | 依据材料或指南版本不清 | 题目答案可能成立，但依据与本研究指定教材/指南体系不一致，或使用过时指南。 | AERA/APA/NCME Standards；NBME Item-Writing Guide；UroEMAS protocol |
| M7 | 病例真实性不足 | 病例可以作答，但临床场景不自然、检查路径不符合常规、症状体征组合牵强，降低临床相关性。 | NBME Item-Writing Guide；Case & Swanson / NBME manual；UroEMAS protocol |
| M8 | A3/A4 子题设计不良但仍可答 | 子题之间缺乏合理递进，前一题直接泄露后一题答案，或后续题不依赖病例演进。 | NBME Item-Writing Guide；Case & Swanson / NBME manual；UroEMAS protocol |
| M9 | B 型题共享选项设计不良但仍可答 | 共享选项过宽、过窄或区分度弱，导致多个小题不能有效鉴别知识点。 | NBME Item-Writing Guide；Case & Swanson / NBME manual；Haladyna et al. 2002 |
| M10 | 来源风格线索明显但未直接泄露 | 语言过度模板化、题干长度异常、病例结构过于机械，可能影响“MAS-like”来源判断。你的探索性分析已计划考察 language_style、stem_length、cognitive_level、distractor_quality 等线索。 | AERA/APA/NCME Standards；UroEMAS protocol |
| M11 | 局部术语不规范且可能影响理解 | 专业术语、疾病名称、检查名称或指南术语使用不规范，虽未造成核心事实错误，但可能影响考生理解。 | NBME Item-Writing Guide；Downing 2005；UroEMAS protocol |

## Technical/Formatting Defects

### T1：选项数量错误

| 错误类型 | 是否致命 | 原因 |
| --- | --- | --- |
| 只有 A–D 四个选项 | 是 | 与预设格式不一致，影响答题和评分 |
| 出现 A–F 六个选项 | 是 | 超出统一题型规则 |
| 缺少某个选项，例如没有 C | 是 | 作答结构不完整 |
| 两个选项都标为 B | 是 | 选项编号不唯一 |
| 选项编号与答案键无法对应 | 是 | 无法可靠评分 |

### T2：正确答案标记与选项格式不匹配

| 错误类型 | 是否致命 | 原因 |
| --- | --- | --- |
| 答案写为 F，但选项只有 A–E | 是 | 答案键无效 |
| 答案写为 “C/D” 但题型是单选题 | 是 | 单选题答案不唯一 |
| 答案写为 “A、C”，但题型标记为 A1/A2 单选题 | 是 | 题型与评分规则冲突 |
| X 型题答案格式缺失，例如未说明多选 | 是 | 考生无法知道作答规则 |
| 答案键为空 | 是 | 无法评分 |
| 答案键与选项编号错位 | 是 | 导致系统性误判 |
