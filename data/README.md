# Data

- `banks/bank_*.json`: original/manual question banks.
- `banks/new_bank_*.json`: MAS-generated question banks with accumulated evaluation fields.
- `raw/datasets/`: raw source documents and text exports used during bank construction.

Do not infer study design or plotting requirements from data filenames. Use `plot/agent_readable/` for plot requirements and `docs/AGENT_GROUNDING.md` for source priority.
